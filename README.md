# Oracle Fusion AI SQL Generator

AI-powered SQL generator for Oracle Fusion Finance using LangChain, RAG, ChromaDB, HuggingFace Embeddings, Groq LLM, and Streamlit.

## Features
- Oracle Fusion SQL generation
- RAG-based metadata retrieval
- Chroma vector database
- Groq LLM integration
- Streamlit frontend

## Run Project
pip install -r requirements.txt
streamlit run app.py
